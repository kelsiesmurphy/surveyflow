<!-- GETTING STARTED -->
## Getting Started
### Prerequisites

To run this app, you must run: 
* npm install
  ```sh
  npm install
  ```

* run in development
  ```sh
  npm run dev
  ```

Note: To stop the server enter ctrl + c in your Terminal
